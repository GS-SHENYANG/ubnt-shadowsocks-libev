# ubnt-shadowsocks-libev

# 简介

本项目是 [shadowsocks-libev](https://github.com/shadowsocks/shadowsocks-libev) 在 UBNT 路由器上的移植。  
缺失的依赖包可以在[这里](https://packages.debian.org/en/)查询下载。  
目前仅支持 EdgeRouter™ X。  
所有 deb 安装包都在发布时的最新 UBNT 固件下测试过。
